Self-defined final projects are required to meet the following minimum requirements:

Student submits a proposal that describes how they intend to use knowledge graphs, Cypher, and GDS algorithms to analyze a self-selected collection of data.
Use self-selected data set(s) to create a knowledge graph.
Define and answer at least 8 graph EDA questions.  For each question, provide your Cypher query and the output returned by Neo4j.
For the deeper analytical component of the project you have two options:

Option 1

Define and answer at least four (4) deeper analytical questions using Cypher queries
For each question, you must provide: (a) your Cypher query in a labeled code cell, (b) a written Markdown explanation of the query logic, describing specifically how the graph traversal works and why you structured the query as you did, and (c) a written commentary section that explains the conclusions you are able to derive from the results. This same requirement applies to Option 2 for the required Cypher queries.

Option 2

Define and answer at least two (2) deeper analytical questions using Cypher queries and at least one (1) question that requires the use of a GDS algorithm. 
If you opt to use a GDS algorithm, for each algorithm used you are required to: (a) create an appropriate in-memory graph projection containing the nodes and relationships relevant to the algorithm you intend to apply, providing a clearly written explanation of your projection design decisions, (b) execute the GDS algorithm using either the stats, stream, or write execution mode as appropriate, (c) provide the complete Cypher/GDS code used for both the projection and the algorithm call in labeled code cells within your notebook, and (d) provide a substantive written commentary in a Markdown cell that explains your choice of algorithm (with explicit reference to the theoretical underpinnings of the algorithm, e.g., whether it handles directed or undirected relationships, how it uses edge weights if applicable, and what its computational complexity implies for large graphs), and that interprets the results specifically within the context of the data you chose to work with.
